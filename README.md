## Features

- Search Users
- List Users
- Show user profile
- Show repositories of a user

## âš™ï¸ How to setup and run the project

### Setting the project

1. Clone the project

Using SSH

```shell
git clone git@github.com:thiagonunesbatista/git-search.git
```

Or using HTTP

```shell
git clone https://github.com/thiagonunesbatista/git-search.git
```

2. Enter in the folder of the project

Using shell

```shell
cd git-search
```

3. Install all the project's dependencies

```shell
npm install
```

### Starting the project

1. Start the project

```shell
npm run serve
```
